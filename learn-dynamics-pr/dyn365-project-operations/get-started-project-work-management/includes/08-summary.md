يساعدك Project Operations علي تبسيط تعقب الوقت وإدارة المصروفات بالإمكانيات الآتية:

- إرسال الوقت والمصروفات بشكلٍ فوري

- تسهيل توافق أعضاء الفريق

- ضمان دقة التقارير والالتزام بسياسة المصروفات

للحصول على مزيد من المعلومات وملفات الفيديو التوضيحية، راجع [التعامل مع إدارة المصروفات في Dynamics 365 Project Operations](/training/paths/work-expense-management/?azure-portal=true).
