في هذه الوحدة النمطية، ستقوم باستكشاف تنفيذ المشروع وتحليل الإمكانات في Microsoft Dynamics 365 Project Operations. تشمل هذه الإمكانات:
 - إدارة هياكل تنظيم العمل

 - التنبؤات والموازنات

 - الفوترة

 - تحليل إمكانات تعقب تكلفة المشروع. 

## <a name="learning-objectives"></a>الأهداف التعليمية

وفي نهاية هذه الوحدة النمطية، ستتمكن من:

- وصف إمكانات تخطيط المشروع.

- وصف إمكانات تنفيذ المشروع.

- وصف هيكل تنظيم العمل.

- وصف إمكانات فوترة المشروع.

- وصف إمكانات تتبع تكلفة المشروع.
