تناقش هذه الوحدة النمطية إمكانات إعداد التقارير المضمنة في تطبيقات التمويل والعمليات. وهي تناقش أيضاً إمكانات تطبيقات التمويل والعمليات مع منتجات Microsoft الأخرى، مثل Microsoft Excel وOutlook وWord وPower BI وSharePoint.

## <a name="learning-objectives"></a>الأهداف التعليمية

وفي نهاية هذه الوحدة النمطية، ستتمكن من:

- وصف إمكانات إعداد التقارير المضمنة.

- وصف خيارات تحليل البيانات.

- وصف خيارات معالجة البيانات مع منتجات Microsoft. 

- وصف حالات الاستخدام وإمكانات التكامل مع منتجات Microsoft الأخرى.
