قد يكون لديك بيانات موجودة يجب استيرادها إلى Dynamics 365 Sales Professional. باستخدام import tool للبيانات، يمكنك إدخال بياناتك بسرعة إلى التطبيق وتكون جاهزة للاستخدام.

إن استيراد البيانات يدعم ملفات القيم المفصولة بفاصلة (CSV) وقوالب Microsoft Excel.

راجع [استيراد البيانات في Dynamics 365 Sales Professional](/dynamics365/customer-engagement/sales-professional/import-data) لمزيد من المعلومات.
