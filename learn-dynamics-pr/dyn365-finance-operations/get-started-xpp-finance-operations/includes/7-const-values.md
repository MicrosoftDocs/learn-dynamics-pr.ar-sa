يمكن استخدام القيم الثابتة لتحديد متغير وتعيين قيمة لا تتغير. يمكنك تعريف متغير ثابت عن طريق إضافة الكلمة الأساسية `const` قبل تعريف متغير. يوضح المثال التالي كيفية جعل المتغير متغيراً ثابتاً. 
```xpp
const str constantVariable = "Value";
```
